## 変更点サマリ

### PR #178 — RecordDetailScreen

| #   | 変更内容                                                                             |
| --- | ------------------------------------------------------------------------------------ |
| 1   | 写真を画面上部に全幅・高さ280で大きく表示                                            |
| 2   | 写真なし時のプレースホルダーを赤ちゃんアイコン（FontAwesome6: baby）に変更           |
| 3   | 日付を写真の左上にオーバーレイ表示                                                   |
| 4   | タイトルを写真直下に大きく（fontSize 22）表示                                        |
| 5   | 記録日時点の月齢バッジを横並びで表示（実月齢・修正月齢・在胎・生後日数）             |
| 6   | 「編集する」ボタンをコンテンツから削除し、ナビバー右上に「編集」テキストリンクを追加 |

### PR #179 — AchievementListScreen

| #   | 変更内容                                                                                             |
| --- | ---------------------------------------------------------------------------------------------------- |
| 1   | ヘッダーを「名前 ＋ 今日の月齢 ＋ 修正月齢バッジ ＋ 生後日数」スタイルに変更（カレンダー画面と統一） |
| 2   | FlatList → SectionList に変更し、月ごとのセクションヘッダーを追加                                    |
| 3   | 各月の写真付き最新記録をフィーチャーカード（大カード）で表示                                         |
| 4   | 写真付きその他の記録に右サムネイルを追加                                                             |
| 5   | 写真なし記録の右側にカメラアイコンのプレースホルダーを追加                                           |
| 6   | 各カードに記録日時点の修正月齢バッジを追加                                                           |

### PR #181 — TodayScreen

| #   | 変更内容                                                          |
| --- | ----------------------------------------------------------------- |
| 1   | 今日の記録カードを横並びレイアウトに変更（右側に72×72サムネイル） |
| 2   | 写真なし記録のカード右側にカメラアイコンのプレースホルダーを追加  |
| 3   | 全カードを同一サイズで表示（フィーチャーカードなし）              |
| 4   | ヘッダー・月齢表示・「画像として保存」ボタン・FABは変更なし       |
