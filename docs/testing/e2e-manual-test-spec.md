### E2E-001 初回起動・プロフィール未設定時の表示

**優先度**: 高
**対象OS**: iOS / Android

**事前条件**
- アプリをクリーンインストールした状態（データなし）

**手順と期待結果**

| No | 手順 | 期待結果 |
|---|---|---|
| 1 | アプリを起動する | カレンダー画面が表示される |
| 2 | ヘッダー部分を確認する | 「プロフィール未設定」と表示されている |
| 3 | 画面下部のFABボタンを確認する | 「＋記録」FABボタンが表示されている |
| 4 | 下部タブの「今日」をタップする | 今日タブ画面に切り替わる |
|   |  | 「プロフィールを作成してください」のメッセージが表示される |
| 5 | 今日タブ内の設定への導線ボタンをタップする | 設定画面またはプロフィール管理画面へ遷移する |
| 6 | バック操作で今日タブに戻る | 今日タブが正常に表示される |
| 7 | 下部タブの「記録一覧」をタップする | 記録一覧画面に切り替わる |
|   |  | 「プロフィール未設定 記録一覧」と表示される |

### E2E-002 プロフィール新規作成（通常出産）

**優先度**: 高
**対象OS**: iOS / Android

**事前条件**
- プロフィールが未登録の状態

**手順と期待結果**

| No | 手順 | 期待結果 |
|---|---|---|
| 1 | 設定タブをタップする | 設定画面が表示される |
| 2 | 「プロフィールを管理」をタップする | プロフィール管理画面へ遷移する |
| 3 | 「新しいこどもを追加」をタップする | プロフィール入力画面が開く |
| 4 | 名前を入力する（例：「テスト太郎」） | 名前フィールドに入力した文字が表示される |
| 5 | 出生日を選択する（例：2ヶ月前の日付） | 選択した日付が出生日フィールドに反映される |
| 6 | 出産予定日は入力しない（空欄のまま） | 出産予定日フィールドが空のままである |
| 7 | 「保存」をタップする | プロフィール管理画面に戻る |
|   |  | 作成したプロフィール（「テスト太郎」）が一覧に表示される |
| 8 | バック操作またはカレンダータブをタップしてカレンダー画面に移動する | カレンダー画面のヘッダーに名前と月齢が表示される |
|   |  | カレンダーの誕生日セルに「誕生日」ラベルが表示される |
|   |  | 当月のセルに月齢（例：「暦 2ヶ月」）が表示される |

### E2E-003 プロフィール新規作成（早産・出産予定日あり）

**優先度**: 高
**対象OS**: iOS / Android

**事前条件**
- プロフィールが未登録または追加可能な状態

**テストデータ例**
- 出生日：4ヶ月前
- 出産予定日：2ヶ月前（＝在胎約32週で出生）

**手順と期待結果**

| No | 手順 | 期待結果 |
|---|---|---|
| 1 | 設定タブをタップする | 設定画面が表示される |
| 2 | 「プロフィールを管理」をタップする | プロフィール管理画面へ遷移する |
| 3 | 「新しいこどもを追加」をタップする | プロフィール入力画面が開く |
| 4 | 名前を入力する（例：「テスト花子」） | 名前フィールドに入力した文字が表示される |
| 5 | 出生日を選択する（4ヶ月前） | 選択した日付が出生日フィールドに反映される |
| 6 | 出産予定日を選択する（2ヶ月前） | 選択した日付が出産予定日フィールドに反映される |
| 7 | 「保存」をタップする | プロフィール管理画面に戻る |
|   |  | 作成したプロフィール（「テスト花子」）が一覧に表示される |
| 8 | カレンダー画面に移動して、出産予定日以降の月を表示する | カレンダーに「修正 〇ヶ月」のラベルが表示される |
| 9 | カレンダーを出生日〜出産予定日の期間に移動する | 出生日〜出産予定日の間のセルに「在胎 〇週〇日」が表示される |
| 10 | カレンダーを修正月齢の表示上限（デフォルト24ヶ月）以降の月に移動する | 修正月齢の表示が終了し、暦月齢のみ表示される |

### E2E-005 プロフィール削除

**優先度**: 中
**対象OS**: iOS / Android

**事前条件**
- プロフィールが2件以上登録済み

**手順と期待結果**

| No | 手順 | 期待結果 |
|---|---|---|

### E2E-006 複数プロフィール作成・切り替え

**優先度**: 中
**対象OS**: iOS / Android

**手順と期待結果**

| No | 手順 | 期待結果 |
|---|---|---|
| 1 | E2E-002の手順に従ってプロフィールA（例：「子A」）を作成する | プロフィールAが管理画面に表示される |

|   |  | プロフィールAの「✓」マークが消える |
| 4 | カレンダー画面に移動する | カレンダーがプロフィールBの年齢情報で更新されている |
| 5 | 記録一覧タブをタップする | 記録一覧がプロフィールBのものに切り替わっている |
