# リトルベビーログ E2E テスト仕様書兼結果報告書（PR #186）

**対象アプリ**: リトルベビーログ
**テスト種別**: 手動 E2E テスト（実機）
**対象OS**: iOS
**作成日**: 2026-05-01
**更新日**: 2026-05-01
**関連Issue**: #186 fix: UIスタイル統一 — ヘッダー・カード・TodayScreen月齢表示の修正

---

## 背景

PR #178/#179/#181 の E2E テスト（2026/5/1 実施）で発見された NG 3 件に対応。

| NG番号 | 発見箇所       | 内容                                                                                       |
| ------ | -------------- | ------------------------------------------------------------------------------------------ |
| NG-1   | CalendarScreen | ヘッダーの修正月齢表示形式（インラインテキスト）が AchievementListScreen（バッジ）と異なる |
| NG-2   | TodayScreen    | 記録カードのタイトルが縦中央寄せで、きろく一覧（上寄せ）と統一されていない                 |
| NG-3   | TodayScreen    | ヘッダーに月齢が表示されておらず、選択日の月齢をどこからも確認できない                     |
