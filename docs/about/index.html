リトルベビーログ

このアプリについて

「リトルベビーログ」は、早産で生まれたお子さまの修正月齢を確認しながら、日々の成長や小さな出来事を記録できるアプリです。

暦月齢と修正月齢を同時に表示し、日々の記録をカレンダー形式で振り返ることができます。

写真を添付した記録も可能で、思い出をひとつずつ残していくことができます。

修正月齢・在胎週数の計算方法について

修正月齢

修正月齢は、出産予定日を起算日として経過した月数・日数で算出します。

出生日から出産予定日までの日数が42日以上（在胎37週0日未満の早産）の場合に表示されます。

出産予定日以降は修正月齢を表示し、設定した表示上限月齢を超えると非表示になります。

在胎週数

在胎週数は、40週0日（280日）を満期として、出生時の在胎日数に出生後の経過日数を加算して算出します。

出生日から出産予定日の前日までのカレンダーセルに表示されます。

表記

  • 暦月齢：誕生日からの経過月数（例：3か月）
  • 修正月齢：出産予定日からの経過月数（例：修正 1ヶ月）
  • 在胎週数：出生時の在胎週数＋出生後経過日数（例：在胎 36週2日）

本アプリで表示される修正月齢・在胎週数は、あくまで計算による参考値です。

データの保存について

本アプリのデータは、利用者の端末内に保存されます。

運営者のサーバーへ送信されることはありません。

端末の故障や紛失、アプリの削除等によりデータが消失する可能性があります。
